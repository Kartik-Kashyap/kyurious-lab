The Math Behind Centrifuge

This is a bounded confidence + homophily + algorithmic bias agent-based model.

1. Opinion Update Rule

For each agent i, we collect opinions from neighbors whose opinions are within the confidence threshold:

\[ \text{If } |o_i - o_j| < \tau \quad \Rightarrow \quad o_i \leftarrow \frac{1}{k} \sum o_j \]

where \(\tau\) = Confidence Threshold, \(k\) = number of similar neighbors.

2. Algorithmic Amplification (The Echo Booster)

\[ o_i' = o_i + (e - o_i) \cdot \alpha \cdot 0.2 \]

where \( e = 1 \) if \( o_i > 0.5 \), else \( 0 \)
\(\alpha\) = Algorithm Strength (0 = neutral, 1 = maximum polarization push)

3. Homophily Network Formation

\[ P(\text{connect } i \text{ and } j) \propto \exp\left( -\frac{|o_i - o_j|}{1 - h + \epsilon} \right) \]

\(h\) = Homophily parameter (higher = stronger preference for similar opinions)

4. Polarization & Clustering Metrics

  • Polarization Index: Based on standard deviation of opinions (scaled to [0,1])
  • Clusters: Number of opinion groups separated by gaps > 0.15
  • Diversity (Entropy): Shannon entropy over 10 opinion bins — lower value = stronger echo chamber
The Emergence Insight
Even modest values of \(\tau\), \(h\), and \(\alpha\) can drive the system from consensus into deep polarization. This is why echo chambers feel "natural" once the reinforcing mechanisms are in place.
