HARMONOGRAPH SIMULATOR

Pendulum Math • Lissajous • Spirograph

How it works:
This simulates two pendulums (X and Y) connected to a pen.
The path is drawn using parametric equations:
x = sin(freqX·t + phase)
y = sin(freqY·t)
with gradual damping to create beautiful decaying patterns.
