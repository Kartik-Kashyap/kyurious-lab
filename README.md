Welcome to the internal research wing of the **Kyurious** multiverse. If **kyurious.kids** is where we explain the world, **Kyurious Labs** is where we try to break it, simulate it, and rebuild it using technology.

## What’s Inside?

Every project here is a **single-file organism**. No bloated frameworks, no 4GB `node_modules` folders—just pure, distilled HTML, CSS, and Logic.

  * **Social Simulators:** Watching how ideas spread (and how echo chambers happen).
  * **Mathematical Art:** Manim-style visuals but interactive and real-time.
  * **Forensic Playgrounds:** Light versions of neuro-forensic logic because crime is a system too.
  * **Game Theory Toys:** Digital arenas to test if humans are actually rational (spoiler: we aren't).

## How to Add a New Specimen

1.  **Code it:** Build a simulator in one HTML file. Make it pretty. Make it weird.
2.  **Drop it:** Toss it into the `/projects` folder.
3.  **Frame it:** Take a screenshot for the `/assets/thumbnails` folder or just use the iframe to use the website as the thumbnail itself.
4.  **Announce it:** Update the grid in `index.html`.
